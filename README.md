# circles
보고서 '두 원의 가중합에 대해'에 첨부한 그래프 파일들을 모아둔 곳입니다.

> 일반적으로 두 원의 교점을 지나는 원의 방정식은 $kf(x,y)+(1-k)g(x,y)=0$으로 나타낼 수 있다.
> 이것은 말하자면 두 원의 방정식에 $k$라는 가중치를 적용하여 더한 가중합(weighted sum)이라 할 수 있는데, 항상 원을 나타내는 것은 아니다.
